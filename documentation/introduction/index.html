引擎简介

一个高度灵活、跨平台的同人游戏引擎，用于创建类似《Undertale》风格的游戏，基于 Godot 引擎构建。突破传统 UT 同人游戏工具的限制，UndertaleMakerX 支持零代码快速开发和 JavaScript 脚本扩展，可一键导出为独立可执行文件（Windows/macOS/Linux）和移动包（APK），使创作者能够从简单的 UT 风格战斗到复杂的自定义游戏逻辑构建一切。

快速入门

环境要求

  • 操作系统：Windows 10+/macOS 11+/Linux（Ubuntu 20.04+）
  • 移动端导出：Android Studio（用于 APK 打包）

安装步骤

  • 从 Releases 页面 下载最新版本软件包
  • 解压软件包（无需安装，绿色软件）
  • 运行 UndertaleMakerX.exe (Windows) / UndertaleMakerX.app (macOS) / UndertaleMakerX.x86_64 (Linux)

开发模式

UndertaleMakerX 支持两种开发模式，以适应从初学者到高级开发者的所有技能水平的创作者。

🧩 无代码模式（适合初学者）

  • 通过配置文件修改游戏逻辑
  • 一键预览和导出，无需编程基础
  • 通过极简操作完美还原经典 Undertale 战斗体验

📜 JavaScript 脚本模式（适用于高级开发者）

  • 基于 JavaScript 互操作性，支持自定义复杂逻辑
  • 暴露完整引擎底层 API，打破“仅限原版 UT 对战”的限制
  • 自定义战斗规则、角色 AI、游戏机制，甚至非 UT 风格的玩法
  • 支持导入自定义脚本和第三方库，具有无限的扩展可能性

跨平台导出

一键导出到多个平台，生成的文件是独立的，无需额外的运行时环境：

  • 桌面平台：Windows (EXE)、macOS (APP)、Linux (x86_64/AppImage)
  • 移动平台：Android (APK)
