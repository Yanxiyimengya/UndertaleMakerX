#  UndertaleMakerX 类参考
UndertaleMakerX 将导出到 JavaScript 端的全部类型集成到 `UTMX` 模块包中。
可以从 `UTMX` 包中导入对应的类型，就可以使用类型。

```javascript
import { Vector2, Color } from "UTMX"; // 引入 UTMX 包

let pos = new Vector2(0, 0);
let col = Color.Red;
```
