# 生命周期对象

生命周期对象指在 UTMX 中，你可以通过编写脚本监听它们创建，每行，销毁事件的一些列对象。

下列列表列出了哪些对象属于生命周期对象:

- Enemy
- Sprite
- BattleProjectile
- DrawableObject

你可以在继承脚本中添加下列方法，他们在合适的时机就会由引擎调用。

### start()

在对象被创建且活跃在场景时触发

###  update(delta: float)

对象活跃在场景时每帧触发，其中该方法传递一个参数 `delta` 表示与上一帧距离的时间差，单位为秒。

### onDestroy()

对象离开场景且被销毁时触发
