# Debugger 模块

Debugger 是 UTMX 框架的调试日志管理类，负责统一输出游戏运行时的日志、警告、错误信息，便于开发阶段排查问题。
通过 `UTMX.debug` 全局对象访问。

<br>

?> Debugger 模块是全局模块，在游戏的任意生命周期阶段都可直接调用。

---

## 日志输出

### print
输出普通日志信息，支持多参数拼接输出。

**返回值** : `void`

| 参数 | 类型 | 可选 | 默认值 | 简介 |
| :----- | :---- | :---- | :---- | :---- |
| message | any | ❌ | - | 要输出的日志内容（可传入多个参数） |

---

### log
`print` 方法的别名，功能完全一致，用于输出普通日志信息。

**返回值** : `void`

| 参数 | 类型 | 可选 | 默认值 | 简介 |
| :----- | :---- | :---- | :---- | :---- |
| message | any | ❌ | - | 要输出的日志内容（可传入多个参数） |

---

### warning
输出警告级别日志信息，通常用于提示非致命性问题。

**返回值** : `void`

| 参数 | 类型 | 可选 | 默认值 | 简介 |
| :----- | :---- | :---- | :---- | :---- |
| message | any | ❌ | - | 要输出的警告内容（可传入多个参数） |

---

### error
输出错误级别日志信息，用于提示致命性问题或异常。

**返回值** : `void`

| 参数 | 类型 | 可选 | 默认值 | 简介 |
| :----- | :---- | :---- | :---- | :---- |
| message | any | ❌ | - | 要输出的错误内容（可传入多个参数） |

---

## 使用示例

```javascript
import { UTMX } from "UTMX";

// 输出普通日志
UTMX.debug.log("玩家等级提升至：", 5);

// 输出警告日志
UTMX.debug.warning("物品 ID 未注册");

// 输出错误日志
UTMX.debug.error("战斗系统初始化失败");
```
