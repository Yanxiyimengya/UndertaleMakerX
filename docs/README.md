# UndertaleMakerX 官方文档

![UndertaleMakerX图标](imgs/icon.svg)

UndertaleMakerX 是一款基于由 [Godot](https://godotengine.org/) 开发的 Undertale 同人游戏引擎。

它主要使用 JavaScript 语言，带有编辑器，并允许你轻松、快速地创作 Undertale 同人游戏。

## 入门

---

关于如何开始使用 UndertaleMakerX 请查阅[入门教程](modules/)

## 贡献

---

UndertaleMakerX 也期望同 Undertale 社区的朋友们同心协力，构建出一款强大的而开发工具，如果你不理解某些内容、在某些地方遇到了问题，请联系作者。

如果你想为 UndertaleMakerX 添砖加瓦，我们非常欢迎你，你可以在 [GitHub 仓库](https://github.com/Yanxiyimengya/UndertaleMakerX/issues)上提交 Issue 或创建 Pull Request，提交新的功能请求、BUG 反馈。
